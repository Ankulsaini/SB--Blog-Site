# ProtoPress Blog Prototype

A lightweight WordPress-style blog prototype built with plain HTML, CSS, and JavaScript.

## Included pages

- **Home (`index.html`)**: post listing feed
- **Single Post (`post.html`)**: detail page loaded by query id
- **Mock Admin (`admin.html`)**: create-post form

## Features

- Light theme UI
- Seeded sample posts
- Browser localStorage persistence
- Reset button on home to restore sample data

## Run locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
